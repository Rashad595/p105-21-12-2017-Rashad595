# P105-21.12.2017

### Tapşırıq
 Tapşırıqlar draw.io-da flowchart şəklində çəkilməli və şəkil formatında bura yüklənməlidir.
 
 - İstifadəçi 0-la 100 arasında bir rəqəm daxil edir və biz bu rəqəmi adı N olan bir dəyişəndə (variable) saxlayırıq. Daha sonra isə istifadəçiyə bu ədədin ya 3-ə, ya 7-ə, ya hər ikisinə, ya da heç birinə qalıqsız bölünüb-bölünmədiyini cavab olaraq göstəririk.
 - İstifadəçi yaşını və yaşadığı şəhəri daxil edir (Azərbaycan şəhərləri). Əgər yaşı 18-dən böyük olarsa və yaşadığı şəhər ya Sumqayıt ya da Bakı olarsa, istifadəçiyə "Siz qeydiyyatdan keçə bilərsiniz deyir", əks halda "keçə bilməzsiniz" deyir.
